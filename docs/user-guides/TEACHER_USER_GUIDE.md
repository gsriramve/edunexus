## 1. Getting Started

### Logging In
1. Visit your college's EduNexus portal
2. Click "Sign In"
3. Enter your faculty email and password
4. You'll be directed to the Teacher Dashboard

### First-Time Setup
1. Complete your profile
2. Verify assigned subjects
3. Review your class schedule
4. Enable notifications

## 3. Attendance Management

### Marking Daily Attendance

1. Go to **Attendance** from menu
2. Select the class/subject
3. Choose the date (defaults to today)
4. Mark each student:
   - **P**: Present
   - **A**: Absent
   - **L**: Late
   - **OD**: On Duty
5. Click "Save Attendance"

### Bulk Attendance
1. For larger classes, use "Mark All Present"
2. Then mark individual absences
3. This saves time for well-attended classes

### Viewing Reports
1. Go to **Attendance** > **Reports**
2. Select class and date range
3. View attendance percentages
4. Export to Excel

### Attendance Alerts
- Students below 75% attendance flagged
- System sends automatic warnings
- Review at-risk students list

## 5. Marks Entry

### Entering Exam Marks

1. Go to **Marks** from menu
2. Select the exam
3. Choose subject and section
4. Enter marks for each student
5. Grades auto-calculate
6. Save marks

### Bulk Marks Entry
1. Click "Bulk Entry"
2. Download template (Excel)
3. Fill marks in template
4. Upload completed file
5. Review and confirm

### Grade Calculation
Grades are calculated automatically:

| Percentage | Grade | Points |
|------------|-------|--------|
| 90-100% | A+ | 10 |
| 80-89% | A | 9 |
| 70-79% | B+ | 8 |
| 60-69% | B | 7 |
| 50-59% | C | 6 |
| 40-49% | D | 4 |
| Below 40% | F | 0 |

### Editing Marks
1. Find the exam entry
2. Click "Edit"
3. Modify marks
4. Add remarks if needed
5. Save changes

## 6. Assignments

### Creating Assignments

1. Go to **Assignments** from menu
2. Click "Create Assignment"
3. Fill details:
   - Title
   - Subject
   - Instructions
   - Due date
   - Max marks
   - Attachments (optional)
4. Select target students/sections
5. Publish

### Managing Submissions
1. Go to **Assignments** > Select assignment
2. View all submissions
3. Download submitted files
4. Grade each submission
5. Add feedback

### Grading Workflow
1. Click on submission
2. Review content
3. Enter marks
4. Add comments/feedback
5. Mark as graded
6. Student receives notification

### Assignment Analytics
- Submission rate
- Average score
- Late submissions
- Non-submissions list

## 7. Course Materials

### Uploading Materials

1. Go to **Materials** from menu
2. Select subject
3. Click "Upload"
4. Choose file type:
   - PDF (notes, papers)
   - PPT (presentations)
   - Video (lectures)
   - Links (external resources)
5. Add title and description
6. Select target audience
7. Upload

### Organizing Materials
- Create folders by topic
- Tag by chapter/unit
- Mark as required/optional
- Set visibility dates

### Supported Formats
- Documents: PDF, DOC, DOCX
- Presentations: PPT, PPTX
- Videos: MP4, MOV (up to 500MB)
- Images: JPG, PNG

## 8. Communication

### Announcements
1. Go to **Communication** > **Announcements**
2. Click "New Announcement"
3. Write message
4. Select recipients (class/section)
5. Choose priority level
6. Publish

### Messaging Students
1. Go to student profile
2. Click "Send Message"
3. Write message
4. Attach files if needed
5. Send

### Parent Communication
1. From student profile
2. Click "Contact Parent"
3. Send message via app
4. View communication history

### Bulk Communication
1. Go to **Communication** > **Bulk**
2. Select recipients
3. Choose channel (App/Email/SMS)
4. Compose message
5. Send or schedule

## FAQ

**Q: Can I edit attendance after saving?**
A: Yes, within 24 hours. After that, contact admin.

**Q: How do I handle late submissions?**
A: Mark as late when grading. The system tracks this.

**Q: Can students see their marks immediately?**
A: Marks are visible after HOD approval (if configured).

**Q: How do I contact multiple parents?**
A: Use bulk communication feature.
