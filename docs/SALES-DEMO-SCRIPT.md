### Recording Tips
1. Use 1920x1080 resolution
2. Enable smooth scrolling
3. Pause 2-3 seconds on each key screen
4. Use voiceover or add text overlays in post
5. Background music: Upbeat, corporate-friendly
6. Add transitions between persona switches

### Recommended Tools
- **Screen Recording**: OBS Studio, Loom, or QuickTime
- **Editing**: DaVinci Resolve (free) or Adobe Premiere
- **Music**: Epidemic Sound, Artlist, or YouTube Audio Library
